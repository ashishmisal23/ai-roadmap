# Code of Conduct

This project follows a simple rule:

**Be respectful. Be helpful. Be professional.**

## Our Standards

We expect everyone to:
- Be kind and respectful
- Use welcoming language
- Accept constructive feedback
- Help beginners without judgement

## Unacceptable Behavior

- Harassment or hate speech
- Personal attacks
- Disrespectful comments
- Spam or self-promotion without value

## Enforcement

If someone violates this code:
- Maintainers may remove comments or contributions
- Repeated issues may result in a ban
