This repository is meant to help learners understand AI step by step, and contributions are always welcome.

## How You Can Contribute

You can help by:
- Adding new **learning resources**
- Improving existing explanations
- Fixing broken links
- Adding beginner-friendly notes
- Improving structure or formatting
- Suggesting better learning order

## Contribution Guidelines

1. **Fork** this repository
2. Create a new branch  
   ```sh
   git checkout -b feature/your-change
   ```
3. Make your changes
4. Commit with a clear message
   ```sh
   git commit -m "Add NLP resources"
   ```
5. Push to your fork
   ```sh
   git push origin feature/your-change
   ```
6. Open a Pull Request

---

## Rules to Follow

- Keep language simple and beginner-friendly
- Prefer free resources
- Avoid duplicate content
- Mention if a resource is advanced
- Do not remove existing content without a reason

## Code of Conduct
Please follow the [Code of Conduct](#/CODE_OF_CONDUCT.md)
