## Data Science Libraries

These libraries help you **work with data**, perform analysis, and create visualizations.

### 1. NumPy

* Used for numerical computing and working with arrays.
* Required by almost every AI/ML library.
* Install:

  ```sh
  pip install numpy
  ```

### 2. Pandas

* Helps with data cleaning, manipulation, and analysis.
* Very useful for working with CSV, Excel, and tabular data.
* Install:

  ```sh
  pip install pandas
  ```

### 3. Matplotlib

* Used to create charts, graphs, and plots.
* Great for understanding data visually.
* Install:

  ```sh
  pip install matplotlib
  ```

### 4. Seaborn

* Built on top of Matplotlib.
* Makes data visualization easier and more attractive.
* Install:

  ```sh
  pip install seaborn
  ```

### 5. SciPy

* Used for mathematical, scientific, and engineering computations.
* Commonly used along with NumPy.
* Install:

  ```sh
  pip install scipy
  ```

---

## Machine Learning Libraries

These libraries are used to **train, evaluate, and optimize ML models**.

### 1. Scikit-learn

* Most popular ML library for beginners.
* Supports classification, regression, clustering, and more.
* Install:

  ```sh
  pip install scikit-learn
  ```

### 2. XGBoost

* Powerful gradient boosting library.
* Widely used in competitions and real-world ML systems.
* Install:

  ```sh
  pip install xgboost
  ```

### 3. LightGBM

* Fast and efficient gradient boosting framework.
* Works well with large datasets.
* Install:

  ```sh
  pip install lightgbm
  ```

### 4. CatBoost

* Gradient boosting library that handles categorical data well.
* Easy to use with strong performance.
* Install:

  ```sh
  pip install catboost
  ```

---

## Computer Vision Libraries

These libraries help machines **understand and process images**.

### 1. OpenCV

* Industry-standard library for image and video processing.
* Used in real-time computer vision applications.
* Install:

  ```sh
  pip install opencv-python
  ```

### 2. scikit-image

* Focused on image processing algorithms.
* Works well with NumPy arrays.
* Install:

  ```sh
  pip install scikit-image
  ```

### 3. Pillow

* Image processing library (modern replacement for PIL).
* Useful for loading, editing, and saving images.
* Install:

  ```sh
  pip install pillow
  ```

## Deep Learning Libraries

These libraries are used to **build and train neural networks**.

### 1. TensorFlow

* End-to-end deep learning framework by Google.
* Used in research as well as production.
* Install:

  ```sh
  pip install tensorflow
  ```

### 2. Keras

* High-level deep learning API.
* Runs on top of TensorFlow and is beginner-friendly.
* Install:

  ```sh
  pip install keras
  ```

### 3. PyTorch

* Popular deep learning framework with dynamic computation graphs.
* Preferred for research and experimentation.
* Install:

  ```sh
  pip install torch torchvision
  ```

### 4. MXNet

* Deep learning framework focused on performance and scalability.
* Used in some large-scale systems.
* Install:

  ```sh
  pip install mxnet
  ```

## How to Install Packages

To install any Python package:

1. Open **Terminal** (macOS/Linux) or **Command Prompt / PowerShell** (Windows)
2. Run:

   ```sh
   pip install <package_name>
   ```
3. Wait for the installation to finish

### Example

```sh
pip install numpy
```

To install **multiple packages together**:

```sh
pip install numpy pandas matplotlib
```
